# Fingerprint Resolution Values

Use this file for exact resolution appendix values used by `fingerInfo`. Field meanings live in [fingerprint-fields.md](fingerprint-fields.md). Browser language values live in [fingerprint-languages.md](fingerprint-languages.md), interface language values live in [fingerprint-interface-languages.md](fingerprint-interface-languages.md), and timezone values live in [fingerprint-timezones.md](fingerprint-timezones.md).

Source: user-provided RoxyBrowser API appendix attachment. Count from this file: 30 resolutions.

## Resolution Values

Format: `Width x Height`. When setting a custom resolution, use `resolutionType: true` and split a listed value into string fields, for example `1920x1080` becomes `resolutionX: "1920"` and `resolutionY: "1080"`.
